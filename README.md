# Company-Data-Analysis
“I performed exploratory data analysis on company data using Pandas to identify revenue trends, profitability, and business insights.”
Create a modern, clean, and professional infographic-style dashboard poster for a "Company Data Analysis" project.

The design should be visually appealing, easy to understand, and suitable for GitHub README and portfolio presentation.

Include the following sections:

1. Title:
"COMPANY DATA ANALYSIS"

2. Subtitle:
"Exploratory Data Analysis to uncover business insights and trends"

3. Key Metrics (use real values):
- Total Revenue: ₹ 8,000M
- Average Profit: ₹ 167M
- Total Employees: 42,700
- Top Company: TCS (₹ 2,200M)

4. Charts:
- Bar chart: Revenue by company (TCS, Infosys, Wipro, HCL, TechM)
- Line chart: Revenue growth (2022 → 2023)
- Profit margin indicators:
  - TCS: 20%
  - Infosys: 18%
  - Wipro: 17.9%
  - HCL: 16.5%
  - TechM: 15.4%

5. Features Section:
- Revenue Analysis
- Profit Analysis
- Employee Analysis
- Trend Analysis
- Business Insights

6. Table (real data summary):
Include companies with revenue, profit, and employees for 2022 and 2023.

7. Key Insights:
- TCS is the highest revenue generator
- Revenue increased from 2022 to 2023
- Profit margins vary across companies
- Workforce increased year-over-year

8. Tech Stack:
Python, Pandas, NumPy, Matplotlib, Seaborn

9. Tools:
Jupyter Notebook, VS Code, Excel

Design Style:
- Clean UI dashboard layout
- Soft gradient or dark modern theme
- Rounded cards and icons
- Professional typography
- Data visualization focused

Make it look like a real business analytics dashboard suitable for students and developers portfolio.
